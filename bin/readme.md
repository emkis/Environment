# Global bin files

All scripts within this directory will be available globally as commands, as they are being exposed into my `PATH` variable. This allows you to run any script from this directory as a command in your terminal.

## Updating global bin files
Whenever I make any changes to one of my scripts within this repository, I need to sync them again with my computer. For that we need go to this repository's root directory and run the following command:

```bash
# I usually navigate like this: z env
./bin/sync-bins
```
